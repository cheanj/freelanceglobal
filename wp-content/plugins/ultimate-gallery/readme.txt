Create easily a nice gallery of videos for your site with youtube and/or flv videos.

== Description ==

Extremely quickly to use, add the url of videos and that's it all. Thumbnails are created automatically. You can add multiple galleries in the same wp. Support for mobile devices.

<a href="http://www.extendyourweb.com/wordpress/ultimate-video-gallery/">Live DEMO and plugin page</a>.

Key Features:

<ul>

<li> Video gallery to play youtube videos or flv files.</li>

<li> Width, height, round and border of the plugin are configurable.</li>

<li> A list of video thumbnails with paging. You can change the number of thumbnails per page.</li>

<li> Player size and size of thumbnails configurable.</li>

<li> Title and description for each video.</li>

<li> Thumbnail image configurable or automatic.</li>

<li> Different designs thumbnails.</li>

<li> Load youtube videos on mobiles like android or apple.</li>

<li> Autoplay configurable.</li>

<li> Dynamic management and very simple to create and manage multiple video galleries in one website.</li>

<li> The plugin installs the "Ultimate Video" widget, which allows you to display a gallery inside a widget.</li>

</ul>

Once installed and activated the plugin go to "Wordpress admin -> settings -> ultimate video" to create and manage your video galleries. You can load the video gallery within a page or post using the code:

`[ultimate 1/] Where 1 is the gallery ID`

You can also use the ultimate widget to load the gallery into a widget.

With the latest update we've added support for mobile devices, such as Android or Apple. From a mobile device, the plugin loads a gallery in jquery that loads youtube videos through the device application.
		

== Installation ==



1. Upload `ultimate-video-gallery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure and create new video gallery in your Wordpress settings: Ultimate video
4. Place [`ultimate` ID /] in your Posts, where ID is the gallery ID.

5. Admin params:

	- Video URL. Write the url of your videos. You can write the url of the flv videos, the url of the videos on youtube(sample: http://www.youtube.com/watch?v=9I9Ar6upx34).
	
	- Video Titles. Write the titles of your videos.

	- Gallery Width. Number indicating the width of the carousel. You can use a number in pixels or percentage. For example: 100% or 400px.

	- Gallery Height. Number indicating the height of the carousel. You must use a number with px. For example 400px.
